# Bes Outkast Society - News magazine

A quarterly News magazine of distinction, exploring the intersection of extreme performance, high culture, and visionary leadership.

## Features

- **Editorial Design**: A sophisticated, "Dark Luxury" aesthetic inspired by high-end print publications.
- **Dynamic Content**: Integrated with Gemini AI for real-time editorial insights and image generation.
- **Responsive Layout**: Optimized for both desktop and mobile reading experiences.
- **Interactive Elements**: Smooth animations and transitions powered by Framer Motion.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **AI Integration**: Google Gemini API (@google/genai)
- **Build Tool**: Vite

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables:
   - Create a `.env` file based on `.env.example`.
   - Add your `GEMINI_API_KEY`.
4. Run the development server:
   ```bash
   npm run dev
   ```
